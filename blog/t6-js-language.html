JavaScript Language

What, when, where, how?

Parentheses ( )

In JavaScript parentheses are used as a part of a function. They will hold the parameters or the arguments being passed into the function.

Function(parameters)

They can be used to hold conditions in a for loop or if statement.

For (i = 0; i < 20; i++ )

And they can be used to denote that you would like to work out part of an equation.

20 + (35 * 4) = 160

Brackets [ ]

Brackets are used to denote that the information inside the brackets is array info - whether creating an array or calling a specific value of an array.

var array = [23, 23, 45]

Square brackets are also used to access properties of an object using ObjectName['PropertyName']

Braces { }

Braces are used to contain the code that will be executed in a function

Function(parameters)_{code to execute}

They also hold object information when using Literal Notation

Var myObject = {propertyName: value}

Single Quotes ' '

Single quotes and double quotes can be used interchangeably to denote that the text/numbers inside the quotes inside are a string. Best practice is to always use single quotes.

Double Quotes " "

If you want to have single or double quotes show up in a string you can use them together - having the opposite quotes

surrounding the string to the quotes you want to show up inside the string.

Double Outside / Single inside the string "Heya 'Pearl'" or Single outside / double inside the string 'I'm "Pearl"'
