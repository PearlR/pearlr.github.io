Parentheses ( )

In JavaScript parentheses are used as a part of a function

Brackets [ ]

Brackets are used to denote that the information inside the brackets is array info - whether creating an array or calling a specific value of an array.

Braces { }

Braces are used to contain the code that will be executed

Single Quotes ' '

Single quotes and double quotes can be used interchangeably to denote that the text/numbers inside the quotes inside are a string.
